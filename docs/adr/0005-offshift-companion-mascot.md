# ADR 0005: Use an original Offshift companion mascot

- Status: Superseded by ADR 0015
- Date: 2026-07-16

## Context

An interruption needs warmth and clarity. The Red Card reference shows how one recognizable ritual can make an action memorable, but its visual assets and interaction should not be copied.

## Decision

Offshift will use an original, compact companion mascot with routine, drift, protect, break-active, and pre-lock states. It is decorative around accessible text actions and clearly presented as an Offshift character, not an official OpenAI or Codex character.

## Consequences

The UI can be visually memorable without hiding the reason for an intervention or depending on an image. Public-brand review remains a release requirement.
